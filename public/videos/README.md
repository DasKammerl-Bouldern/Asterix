# Video für 404-Seite

Lege deine **MP4-Datei** hier ab und nenne sie **`404.mp4`**.

- Pfad im Projekt: `public/videos/404.mp4`
- Im Browser wird sie unter `/videos/404.mp4` geladen.

Du kannst die Datei per Drag & Drop in diesen Ordner kopieren oder im Finder/Explorer:
**Asterix → public → videos → Datei „404.mp4“ hier ablegen.**
